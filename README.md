# Sasaya Vim

## Installation

1. Clone `.vim` to your home directory.

    ```bash
    git clone https://github.com/storyn26383/sasaya-vim.git ~/.vim
    ```

2. Make a symbolic link to vimrc.

    ```bash
    ln -s ~/.vim/.vimrc ~/.vimrc
    ```

3. Install Plugins

    ```bash
    vim +PlugInstall +qall
    ```
